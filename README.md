# FlowPay Fraud Risk SQL Analysis

## Project Overview
This project simulates a fintech startup (FlowPay) and analyzes transaction data to detect fraud patterns, risky users, and platform performance using SQL.

## Objectives
- Investigate fraud trends
- Analyze payment system performance
- Identify high-risk merchants
- Detect suspicious user behavior
- Build a fraud risk scoring model

## Dataset Scale
Users: 25,000  
Merchants: 3,000  
Devices: 40,000  
Transactions: 2,400,000
